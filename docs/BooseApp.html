Circle

it declares that public class Circle that inherits from BOOSE.CommandOneParameter a BOOSE base class. this shows that Circle is a command that takes one parameter (the radius). it allows the command to integrate into the BOOSE interpreter system.

Main application form for the BOOSE interpreter UI. HOSTS the program in out textbox, a run button, an error log and a picture box that dispalyes the drawing canvas.

MoveTo

Rectangle
