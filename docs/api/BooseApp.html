AppInt

Minimal interpreter for the small BOOSE-like script shown by the user.

AppMethods

Interpreter extended with simple method definitions and calls. Supports method declaration form: method int Name int p1, int p2 Name = <expression> end method and invocation: call Name arg1 arg2

The call evaluates the method body in a local scope and stores the result in a global variable named after the method so subsequent "write Name" or other expressions can reference it.

Implementation reuses most of the AppFor language features (loops, expressions, commands).

AppReal

Simple real-number interpreter for a small BOOSE-like script.

AppWhile

Interpreter that supports integer variables, assignments and structured while/end while loops, plus basic drawing commands: pen, moveto, circle, rect and nested if/else/end if inside blocks. Implements IInterpreter.

Array

Minimal interpreter that supports:

  • int and real variable declarations
  • array declarations: "array int name size" / "array real name size"
  • poke: "poke arr index = expr" (assign into array)
  • peek: "peek var = arr index" (read from array into variable)
  • pen, moveto, circle, write This class is intentionally small and deterministic for UI use.
